介绍

Scoop 是 Windows 上的命令行安装程序

特点如下：

  • 大多数程序安装不需要管理员权限
  • 自动设置环境变量
  • 命令行操作，类似 macOS 下的 HomeBrew
  • 常见开发环境都可以轻松搞定
